1. Chatbot ( sedang di kerjakan )

2. Speech To Text ( selesai )

3. QrCode ( akan datang )

4. Text to Speech ( akan datang )
5. Memainkan audio ( akan datang )
6. Memainkan video ( akan datang )
7. Menghandle banyak account social media
